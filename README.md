# Hearts Score Keeper

A mobile-first 2-page Hearts scoring app.

## Files
- `index.html` — setup page
- `game.html` — live scoring page
- `README.md`
- `.nojekyll`

## Features
- 3 to 5 players
- target score
- clean short score inputs
- blank score fields by default
- pass direction and cards to pass
- live game timer
- live round timer
- scoreboard with round times
- shoot the moon buttons
- confetti winner celebration
- final ranking by place with final scores
